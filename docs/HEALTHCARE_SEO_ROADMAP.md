# Healthcare SEO Roadmap

SEO is downstream of evidence. This roadmap organizes durable executive search intent around published operating knowledge; it is not permission to create pages for every healthcare keyword or claim domain expertise without proof.

## Sequencing and quality gate

1. Complete the P0 diagram and guide packages before creating any new hub page.
2. Each indexable page requires: an approved evidence ID; one diagram or assessment artifact; an explicit claim boundary; a link to an existing assessment; and no unsupported metric or named-program claim.
3. Build P1 pages only after the specified composite/review artifact is admitted to the Evidence Inventory.
4. Keep P2 terms out of healthcare service navigation and conversion copy. Capture them as sales-discovery questions instead.
